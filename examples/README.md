# MARS — Example Casts

These examples show the **range** of MARS: the same world, the same verbs, two very
different purposes. Each is a set of **rooms** (contexts) and a **cast** (roles defined
by skills). Copy a skill into `.opencode/skills/<name>/SKILL.md` to use it.

| Example | What it is | Cast |
|---------|------------|------|
| [Dungeon Master](dungeon-master/) | a text-adventure game | Dungeon Master, wizard, scholar, merchant |
| [Scrum Team](scrum-team/) | a virtual software team | Product Owner, Scrum Master, 2 seniors, 4 juniors |

The point: a "room" is an abstract boundary — a dungeon *or* a sprint board. An "avatar"
is a role — a wizard *or* a Product Owner. The MUD skin and the workplace skin are the
same machine underneath.
